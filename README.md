ロボットシステム工学課題
# clock.py
- このプログラムは、ROS 2（rclpy）を用いて現在の時刻を1秒周期で「何時何分何秒（HH:MM:SS）」形式で表示します。
- 結果は画面上に書き出されます。
## 使用方法
- １．ビルド(colcon build)
- ２．設定(source ~/.bashrc)
- ３．実行(ros2 run mypkg_2 clock)
- 実行結果(例)：[INFO] [1767152894.613351273] [clock_display]: 現在の時刻: 12:48:14
- 　 　　　   ：[INFO] [1767152895.590292123] [clock_display]: 現在の時刻: 12:48:15
## ライセンス
